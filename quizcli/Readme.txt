Functions available.

1. Get List of question Ids: 
                go run main.go questions

2. Get Question & List of Possible Answers (Last digit is the question Id): 
                go run main.go question 2               

3. Results List of 7 answers: 
                go run main.go results 1 1 1 1 6 7 8
        



There is also one function that takes you through all the process 
                go run main.go startquiz
